Potions Description

Enhanced potion system with detailed descriptions for every potion in-game

Java Edition Resource Pack Under Review

Description

This resource pack enhances Minecraft's potion system by providing detailed descriptions of every potion directly in-game. Whether you're new or experienced, it helps you make better choices while brewing your potions.

Features

  • Clear descriptions for all potions, splash potions, lingering potions, and tipped arrows
  • Brewing ingredients with helpful tooltips explaining their purpose
  • Clean interface without confusing duration timers
  • Beginner-friendly explanations that focus on what each potion does
  • Complete coverage of all vanilla Minecraft potions including newer effects
  • Multi-platform support (Data Pack, Fabric, Forge, NeoForge, Quilt)

Perfect For

  • New players learning the brewing system
  • Experienced players who want quick reference
  • Educational servers and tutorials
  • Players who want to understand potion mechanics
  • Anyone tired of consulting external wikis
