FAQ

Common questions from players browsing the catalog

Are the download links on this site direct official sources?

Yes. Project cards open a dedicated detail page on this site, and the primary action links point to the official Modrinth listing or the official creator profile when a public listing is not available.

Can I browse only Java Edition or only Bedrock projects?

Yes. Use the edition filter chips to switch between all releases, Java Edition projects, or Bedrock add-ons without losing the current search flow.

What types of Minecraft releases are included here?

The catalog includes mods, data packs, resource packs, shaders, and Bedrock add-ons focused on quality-of-life improvements such as mining, visibility, HUD overlays, and faster building workflows.

Can I search by mechanic instead of the exact project name?

Yes. The search field matches project titles and descriptions, so terms like vein miner, night vision, HUD, or dynamic lighting will surface relevant results.
