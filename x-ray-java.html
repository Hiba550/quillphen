Java Edition Resource Pack 18.1K Downloads

Description

An X-Ray resource pack for Minecraft Java Edition that highlights ores like diamond, gold, and redstone while making most blocks transparent, allowing players to easily locate valuable resources underground.

Features

  • Highlights valuable ores (diamond, gold, iron, redstone, etc.)
  • Makes stone and other common blocks transparent
  • Easy to install and use
  • Compatible with most Minecraft versions
  • Optimized for underground mining

Installation

  1. Download the resource pack from one of the links below
  2. Open Minecraft and go to Options > Resource Packs
  3. Click "Open Pack Folder"
  4. Place the downloaded file in the resource packs folder
  5. Go back to Minecraft and activate the resource pack
