Important: This mod requires a Gemini API key from Google AI Studio. Without it, AI functionality will not work.

  1. Visit Google AI Studio
  2. Log in with your Google account and create an API key
  3. In Minecraft, press K to open configuration menu
  4. Paste your API key and choose a Gemini model

Note: Google offers a free usage tier suitable for most Minecraft players.

Key Features

  • AI Chat: Get Minecraft-specific help and information through natural conversation
  • Natural Language Commands: Describe what you want and let AI generate the command
  • Survival Protection: Commands not allowed in survival mode are automatically ignored
  • Simple Configuration: Easy setup via in-game interface (K key)
  • Multiple AI Models: Supports Gemini 2.5 Flash and Pro models
  • Universal Compatibility: Works with Fabric, Forge, NeoForge, and Quilt

Configuration

  • Press K in-game to open settings menu
  • Paste your Gemini API key
  • Select your preferred model:
    • Gemini 2.5 Flash Preview (Faster responses)
    • Gemini 2.5 Pro Preview (More detailed responses)
    • Gemini 2.0 Experimental (Latest features)
  • Customize keybindings if needed

Installation

  1. Download the correct version for your mod loader
  2. Place the mod .jar file in your mods folder
  3. Launch Minecraft with your preferred loader
  4. Press K and configure your API key
  5. Start chatting with AI!
